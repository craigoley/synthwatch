## EXECUTED ‚Äî CodeQL `js/insecure-temporary-file` (#4, #5)

**Site:** `runner/scripts/preview-email.mjs:66,67` (writes) via `:18` `const OUT =
'/tmp/synthwatch-email-samples'` + `:19` `mkdirSync(OUT)`.

**TRUE POSITIVE.** A fixed, world-predictable path in the shared temp dir (`/tmp/synthwatch-email-samples`)
is created then written into ‚Äî the classic symlink/clobber/race the rule flags: another user (or a
planted symlink) can pre-create the path and redirect or read the writes.

**Blast radius: minimal.** This is a **dev-only styling playground** (`:2` "Render-check / styling
playground ‚Ä¶ Pure render ‚Äî sends nothing"), run by hand (`node scripts/preview-email.mjs`). It is
**not** in the deployed runner image's runtime path ‚Äî no prod behavior, no schema, no guard/shim.

**Fix (this PR):** replace the predictable path with `mkdtempSync(join(tmpdir(), 'synthwatch-email-samples-'))`
‚Äî a private, randomized dir (mode 0700, unique suffix); drop the now-unused `mkdirSync`; the script
prints the dir on exit. Proof the pattern is gone:
```
$ grep -nE "mkdirSync|/tmp" runner/scripts/preview-email.mjs   ‚Üí NONE
$ node scripts/preview-email.mjs
wrote 4 samples to /var/folders/.../T/synthwatch-email-samples-RuXral/*.{html,txt}   # randomized suffix
```
`tsc --noEmit` = 0, `eslint` = 0.
